How to Automate Scheduled Ad Reporting

The ProblemThe AI Solution
Manually downloading sheets from Tableau, deleting columns, and copy-pasting into Google Sheets every morning is a high-effort, repetitive task. Use Gemini in Google Sheets to automate the cleaning, formatting, and analysis. A single prompt can replace all the manual deletion and color-coding.
Primary Roles: Media Planners Tools: Gemini in Sheets, Google Sheets Difficulty / Phase: Phase 2: Workflow Automation

Step-by-Step Training Workflow

Step 1: Paste Your Raw Data

Create a Google Sheet with two tabs: Raw Data and Clean Report. Copy and paste the *entire* uncleaned dataset from Tableau into the Raw Data tab.

Step 2: Use Gemini to Clean and Format

Go to your Clean Report tab. In cell A1, write a prompt to do all the work:

=GEMINI("Read the 'Raw Data' tab. 
1.  Keep only these columns: [Column A Name], [Column C Name], [Column F Name].
2.  Delete all other columns.
3.  Create a new column called 'Status'.
4.  If [Column C Name] is 'Completed', set Status to 'Done'.
5.  If [Column F Name] is blank, set Status to 'Assets Needed'.
6.  Otherwise, set Status to 'In Progress'.")
